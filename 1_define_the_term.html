Define the terms: Website, Webpage, Web browser, Web server, HTML, CSS

What is Website?

A website is a collection of many web pages, that contains text, images, video, audio or other media that shared by a common domain name and published on at least one web server.

What is Webpage?

A webpage is a document written in HTML and can be viewed on any web browser.

Webpage can be accessed by entering the URL for that web page, and once it is loaded, it appears on the user's web browser.

Each webpage is linked with a unique URL.

What is Web browser?

A web browser is an application for accessing websites.

When a user requests a web page from a particular website, the browser retrieves its files from a web server and then displays the page on the user's screen.

What is Web server?

A web server is software and hardware that uses HTTP (Hypertext Transfer Protocol) and other protocols to respond to client requests made over the World Wide Web.

The main job of a web server is to display website content through storing, processing and delivering webpages to users.

What is HTML?

HTML stands for Hyper Text Markup Language.

HTML is the standard markup language for creating Web pages.

HTML describes the structure of a Web page.

HTML consists of a series of elements.

HTML elements tell the browser how to display the content.

HTML elements such as "this is a heading", "this is a paragraph", "this is a link", etc.

What is CSS?

CSS stands for Cascading Style Sheets

CSS describes how HTML elements are to be displayed on screen, paper, or in other media

CSS saves a lot of work. It can control the layout of multiple web pages all at once
